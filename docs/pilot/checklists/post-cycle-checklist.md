# Post-Cycle Checklist

Complete this after the reflection window closes.

## Artifacts and data

- [ ] Canonical cycle record retained
- [ ] Exports generated and stored
- [ ] Telemetry trace retained
- [ ] Audit trace retained
- [ ] Feedback records retained

## Notes and anomalies

- [ ] Operator anomaly log completed if needed
- [ ] Failure recovery log completed if needed
- [ ] Prompt drift or schedule drift documented if relevant
- [ ] Known missingness or participant dropout recorded

## Evaluation follow-through

- [ ] Candidate interviewees flagged if qualitative follow-up is planned
- [ ] Follow-up notes location recorded
- [ ] Missingness log updated if feedback is incomplete

## Closeout

- [ ] Cycle status confirmed complete or archived as intended
- [ ] Replay remains possible from retained artifacts
- [ ] Next-cycle carryover risks noted
